🚀 QUICK START (3 STEPS)
───────────────────────────────────────────────────────────

1️⃣  Install Dependencies
   $ cd portfolio-complete
   $ npm install

2️⃣  Start Development Server
   $ npm run dev
   → Opens at http://localhost:5173

3️⃣  Customize Content
   → Edit src/data/portfolioData.js
   → Replace public/resume.pdf with your resume

⚙️ AVAILABLE COMMANDS
───────────────────────────────────────────────────────────
  npm run dev      → Start development server
  npm run build    → Build for production (creates /dist)
  npm run preview  → Preview production build

🎨 CUSTOMIZATION GUIDE
───────────────────────────────────────────────────────────

1. CONTENT (Easy - No coding required)
   File: src/data/portfolioData.js
   - Update your name, email, links
   - Edit about section
   - Add your projects
   - List your experience
   - Update skills, certifications, achievements

2. COLORS & THEME
   File: src/styles/global.css
   Change CSS variables:
   --accent-primary: #3b82f6;    (Your brand color)
   --bg-primary: #0a0a0a;        (Background)
   --text-primary: #f5f5f5;      (Text color)

3. FONTS
   File: index.html
   - Replace Google Fonts link
   - Update CSS variables in global.css

4. LAYOUT & STYLING
   Files: src/styles/*.css
   - Each section has its own CSS file
   - Modify as needed

🌐 DEPLOYMENT OPTIONS
───────────────────────────────────────────────────────────

OPTION 1: Vercel (Recommended - Easiest)
  1. Push code to GitHub
  2. Visit vercel.com
  3. Import repository
  4. Auto-deploys in seconds!
  → Your site: yourname.vercel.app

OPTION 2: Netlify
  1. Build command: npm run build
  2. Publish directory: dist
  3. Deploy!
  → Your site: yourname.netlify.app

OPTION 3: GitHub Pages
  1. Build: npm run build
  2. Deploy /dist folder to gh-pages branch
  → Your site: username.github.io/portfolio

OPTION 4: Custom Hosting
  1. Run: npm run build
  2. Upload /dist folder to your hosting
  3. Done!

🔧 COMMON ISSUES & FIXES
───────────────────────────────────────────────────────────

Issue: npm install fails
Fix: Delete node_modules and package-lock.json, try again

Issue: Port 5173 already in use
Fix: npm run dev -- --port 3000

Issue: Slow build times
Fix: Vite is already optimized, check your system

Issue: Styling not applied
Fix: Check CSS import paths are correct
